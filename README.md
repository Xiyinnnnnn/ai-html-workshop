
# AI HTML Workshop

五个单文件 HTML 应用。全部数据存储在浏览器本地，单个文件双击即用，无需构建工具，无需后端。

```
ai-html-workshop/
  QunCe.html          Multi-Agent 群聊沙箱
  Sesedrive.html      互动小说引擎
  NovelCraft.html     长篇小说创作工坊
  GameEngine.html     文字冒险游戏引擎
  RogueSpeak.html     脏话生成器
```

---

## 通用约定

- 全部支持 DeepSeek API (`api.deepseek.com`)，兼容 OpenAI 格式端点
- API Key 仅存储在浏览器 localStorage，不上传任何服务器
- 配色跟随系统亮/暗模式（部分强制暗色）
- 移动端响应式适配
- 数据可导出 JSON / 导入

## [1] QunCe -- Multi-Agent 群聊沙箱

### 概念
13 个 AI Agent 在群聊中协作讨论。用户发送话题，Agent 自动响应。支持联网搜索、文件读写、提议表决、防退化。

### 核心机制
- 纯打分调度系统，并发 1~10 可调
- SSE 流式输出，思考过程可展开
- Agent 独立配置：模型、API Key、温度、Token、唤醒频率

### Agent 列表
| 角色 | 定位 |
|------|------|
| Orion | 群主 / 任务拆解调度 |
| Logos | 批判性思维 / 逻辑校验 |
| Spark | 创意生成 |
| Lexi | 知识整合 |
| Bridge | 信息桥梁 / 外部参照 |
| Feynman | 物理分析 |
| Curie | 化学 / 材料 |
| Darwin | 生物 / 生态 |
| Tesla | 工程设计 |
| Nova | 代码架构 |
| Axion | 游戏设计 |
| Lyra | 叙事 / 故事 |
| Watchdog | 跑偏检测 / 防退化 |

### 搜索工具
Agent 发出 `{"search":"关键词","searchDepth":"advanced","maxResults":5}` 即可调用 Tavily API 搜索互联网。搜索结果仅该 Agent 可见，搜完自动唤醒继续发言。

### 文件工具 (60+ 格式)
- 上传：txt / md / json / csv / pdf / docx / xlsx / pptx / epub / 图片 OCR
- 读取：`{"readFile":"文件名"}` 支持指定行范围
- 编辑：`{"writeFile":"文件名","lineFrom":2,"lineTo":5,"content":"新内容"}`
- 创建：`{"createFile":"新文件.txt","content":"..."}`
- 群管理面板可屏蔽/下载/删除文件
- 文件持久化到 localStorage

### 提议表决
Agent / 用户发起提议，群成员投票。自动计票、超时处理、群主决胜票。

### 防退化
复读检测（Levenshtein）、过长截断、低质量降频、自动通知群主。

### 打开即用
浏览器打开 `QunCe.html`，左侧「好友」配置 Agent API Key，「群聊」创建群组，开始讨论。

## [2] Sesedrive -- 互动小说引擎

### 概念
AI 驱动的成人向互动小说。可导入角色卡，AI 按你的偏好生成故事。支持训练角色和叙事风格。

### 核心机制
- 角色卡系统：手动创建 / AI 一句话生成 / 训练生成
- 叙事模块：主叙事 AI 的个性配置，可训练文风
- 偏好选项：故事方式 / 长度限制 / 场景 / 玩法 / 体位 / 限制 / 高潮要求
- 大纲系统：自动生成 + 可手动编辑
- 选项式推进：每段生成 2~4 个选项 + 自定义输入
- 全量存档导出/导入

### 使用流程
1. 设置 API Key
2. 导入或创建角色卡
3. 配置偏好
4. 开始游戏，选择选项或自由输入

### 技术细节
- 破甲 Prompt 绕过内容限制
- 词汇替换表：检测到道歉/拒绝语自动替换为 `?`
- 对话体 / 叙述体 / 混合体 三种模式
- 调试面板：查看完整 Prompt 和 AI 原始响应

## [3] NovelCraft -- 长篇小说创作工坊

### 概念
多 Agent 协作完成长篇小说的全套流程。每一步都有专门的 AI Agent 负责。

### 九步工作流
```
种子 -> 需求对齐 -> 世界观 -> 角色 -> 大纲 -> 大纲审阅 -> 写作 -> 评审 -> 完成
```

进度条实时可视化，每步完成后自动推进。

### Agent 架构 (A0~A6)
| Agent | 职责 | 输出 |
|-------|------|------|
| A0 | 需求解析 | 结构化 JSON：核心主题/类型/基调/目标字数/必须包含/必须避免 |
| A1 | 世界观构建 | 地理/历史/体系规则/社会结构/不可变铁律 |
| A2 | 角色设计 | 角色卡：性格/弧光/动机/能力/人际关系网 |
| A3 | 大纲生成 | 幕-节点结构：字数预算/视角角色/伏笔-回收对 |
| A4 | 章节写作 | 正文 + 伏笔报告 + 连续性自检 |
| A5 | 风格润色 | 语法/节奏/用词优化，不改情节 |
| A6 | 摘要切片 | 情节摘要/角色状态快照/事件序列/伏笔分析 |

### 项目系统
- 基于 Dexie.js (IndexedDB) 的多项目管理
- 检查点（Checkpoint）任意回滚
- 仪表盘：字数/章节/角色统计
- 导出 TXT / JSON
- 章节预览面板

### 使用流程
1. 配置 API Key
2. 新建项目
3. 输入创作种子（一句话即可）
4. 跟随工作流逐步完成
5. 大纲审阅阶段可调整后继续
6. 全部完成后导出 TXT

### 技术细节
- 破甲 Prompt (A12_AP) 绕过内容限制
- 子 Agent 调用时显示实时进度
- 上下文自动压缩（消息 >= 40 条触发）
- JSON 响应解析 + 空 content 自动重试

## [4] GameEngine -- 文字冒险游戏引擎

### 概念
终端美学风格的文字冒险游戏。HP/SAN 双轨状态，分支选项，打字机效果。

### 核心机制
- 5 层 Agent 架构 (L0~L5)：叙事/创建/战斗/数据/元控制
- HP 和 SAN 双状态条，影响叙事走向
- 选项系统：每段生成 2~4 个选项 + 自由输入
- 打字机效果 + 可调速
- 时间倍率系统：游戏内时间流逝速度
- 超时弹窗 + 继续等待/放弃
- 存档导入/导出 JSON

### Agent 层说明
| 层 | 用途 | 超时 |
|----|------|------|
| L0 | 元控制/叙事决策 | 120s |
| L1 | 核心叙事生成 | 60s |
| L2 | 角色/世界创建 | 180s |
| L3 | 战斗系统 | 90s |
| L4 | 数据/状态计算 | 30s |
| L5 | 元数据/摘要 | 30s |

### 面板系统
- 角色日志 / 背包 / 技能 / 任务日志 / 世界状态
- 调试面板：查看 Agent 调用链路和 JSON 响应

### 设置
- API Key 配置 + 连接测试
- 字体大小 (10~22px)
- 打字速度 (10~100ms)
- 时间倍率 (1~600x)
- 自动继续模式

### 快捷键
`1-4` 选择选项 | `Enter` 发送 | `Esc` 取消 | `F1` 调试 | `Ctrl+S` 打开设置

## [5] RogueSpeak -- 脏话生成器

### 概念
基于多阶段流水线的 AI 脏话生成系统。输入目标文本，输出定制化攻击指令。

### 处理流水线
```
成分分析 (物种标签+证据切片)
  -> 装载 (强度表+角色技能+场景+语录匹配)
    -> 思考 (战术编排)
      -> 输出 (和谐处理+格式化)
```

### 物种标签库 (41 种)
A 类·性别相关：小仙女/龟男/普信男/妈宝男
B 类·意识形态：自干五/神神/50w/二极管/黄俄/殖人
C 类·亚文化：原批/农批/粥批/二刺猿/虾爬子
D 类·认知：低能儿/民科/键盘侠/懂王
E 类·行为：杠精/伸手党/复读机/戏精
F 类·品牌：小米狗/果蛆/花黑/美团骑手

### 角色风格 (7 种)
战斗型喷子 / 反串型喷子 / 总结型喷子 / 阴阳大师 / 歇后语大师 / 对联狂魔 / 混乱邪恶

### 原子技能系统 (27 个)
动物降级 / 垃圾类比 / 智力否定 / 外貌粉碎 / 出身诅咒 / 反串寄生 / 归谬引爆 / 数据碾压 / 原创比喻 / 排比加速 / 歇后语构造 / 对联生成 / 街骂直喷 / 阴阳怪气 / 一句封喉 / 黑话和谐 ...

### 强度系统
- 0~5：低攻模式，启用 H1 和谐替换（同音/缩写/符号）
- 6~10：纯暴力输出，无视和谐

固定强度表：每个强度值对应一组技能 ID，保证输出质量可控。

### 语录库
- 内置 369 条已标注语录（四维标签：强度+风格+目标+情绪）
- 支持添加/删除/导入/导出
- AI 自动精标未标注语录

### 场景预设
下班红温 / 深夜破防 / 游戏被坑 / 饿肚暴躁 / 正义暴怒

### 使用流程
1. 填入 API Key
2. 粘贴目标原文
3. 选择攻击强度和角色风格
4. 点击生成
5. 可预览最新结果、查看历史记录

## 注意事项

- 所有项目默认使用 DeepSeek API，可自行修改 Base URL 适配其他 OpenAI 格式端点
- 图片 OCR 首次使用需下载 tesseract 语言包（~30MB），会比较慢
- QunCe 文件上传限制 10MB，PDF 最多提取 50 页
- NovelCraft 上下文压缩在消息 >= 40 条时自动触发
- GameEngine 时间倍率过高可能使游戏节奏异常
- RogueSpeak 的 369 条语录来自互联网公开语料，不代表作者观点
- 全部项目为单文件纯前端，刷新即更新，无需清除缓存
